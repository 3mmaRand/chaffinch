# chaffinch
[html Output](https://3mmarand.github.io/chaffinch/example-html.html)

[pdf Output](https://3mmarand.github.io/chaffinch/example-pdf.pdf)
by adding to the YAML header

```
format:  
  pdf:  
    documentclass: article  
    papersize: A4  
    mainfont: Calibri  
    geometry:  
      - top=20mm  
      - left=25mm  
      - right=25mm  
```
